# Simple Kanban board
> Created just to test some new libs -
> Recoil and Linaria


## Features
- Add new tasks
- Change task's title, description, tags, status and priority
- Drag and drop them
- Some responsiveness included

## Libs used
- [React] ( + CRA)
- [Recoil] (new state manager by meta)
- [Linaria] (zero runtime CSS-in-JS library)
